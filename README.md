# Legacy Contribution Service Migration

This project demonstrates migration of a legacy VB.NET ADO.NET service to a modern .NET Core backend.

## Legacy Issues
- VB.NET language
- ADO.NET with raw SQL
- Hardcoded connection strings
- No test coverage

## Migration Approach
- Converted VB.NET logic to C#
- Replaced ADO.NET with EF Core
- Implemented clean architecture
- Exposed REST APIs
- Added unit tests

## Tech Stack
- .NET Core 8/9
- C#
- Entity Framework Core
- SQL Server

## API
- POST /api/contributions
